# Task 2: Churn Count Visualization

## Objective

The objective of this task is to visualize the distribution of churned and non-churned customers using a column chart and identify key customer retention insights.

---

## Churn Statistics

| Churn Status | Count | Percentage |
| ------------ | ----- | ---------- |
| Yes          | 1869  | 26.53%     |
| No           | 5174  | 73.47%     |

---

## Activities Performed

### 1. Churn Count Analysis

* Calculated the number of churned customers.
* Calculated the number of active customers.
* Determined churn and retention percentages.

### 2. Column Chart Creation

* Created a Clustered Column Chart.
* Added data labels to display exact counts.
* Added chart title and axis labels.
* Formatted the chart for better readability.

### 3. Business Interpretation

* Analyzed customer retention performance.
* Identified the need for deeper churn investigation.

---

## Key Insights

1. Approximately 26.53% of customers have churned.
2. Approximately 73.47% of customers remain active.
3. The number of active customers is significantly higher than the number of churned customers.
4. Customer retention is relatively strong, but churn remains an important business concern.

---

## Business Question

What factors contributed to the churn of 1,869 customers?

---

## Possible Factors to Investigate

* Monthly Charges
* Contract Type
* Payment Methods
* Customer Support Services
* Internet Service Type
